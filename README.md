## 📌 About
Interactive web application for training Hidden Markov Models using the Baum-Welch algorithm. Visualizes parameter learning in real-time.

## ✨ Features
- Clean web interface with glassmorphic UI
- Real-time visualization of:
  - Transition & Emission matrices
  - Likelihood & Log-likelihood convergence graphs
  - State transition diagrams
  - Alpha, Beta, Gamma values
- Auto-detection of observation symbols
- Matplotlib-powered graphs

## 🛠️ Tech Stack
- **Backend:** Python, Flask, NumPy
- **Frontend:** HTML5, CSS3, Jinja2
- **Visualization:** Matplotlib
